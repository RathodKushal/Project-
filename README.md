Project Title

Simple Interest Calculator (Bash Script)

 Description

This project is a simple Bash script that calculates Simple Interest based on user input. The user is asked to enter the principal amount, rate of interest, and time period. The script then computes and displays the simple interest.

Formula Used

Simple Interest (SI) is calculated using the formula:
